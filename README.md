# Singer Chantepie

Singer Chantepie front end app

## Project setup

```
npm install
```

### Start project in development mode

```
npm run dev
```

### Start project in production mode

```
npm start
```

### Build project for production

```
npm run build
```

### Format project code

```
npm run format
```

### Run eslint checks

```
npm run check
```

### Live link

Check the [project live](https://singer-chantepie.netlify.app) on netlify.
